- I learned that a form field is an area where a user can enter information.
- I learned that a `<label>` describes a form field. The label's `for` value must match the field's `id` to connect them.
- I learned that a CSS `:hover` state changes an element when the user's pointer is over it.
- I learned that the action attribute tells a form where to send its information, while the method controls how the information is sent.
- I learned that client work should begin with an interview so the website is based on the client's needs instead of only my own design preferences.
- I learned how to organize a larger website with multiple HTML pages that share one external CSS stylesheet and consistent navigation.
- I learned that responsive design and readable font sizes are especially important when the intended users include older family members.

- **What I asked**: I asked AI to explain where hover and focus states were being used in my CSS.
