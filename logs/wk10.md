## AI interactions

### Interaction 1

- **What I asked**: I asked AI to read my final project proposal, ask me questions about it one at a time, and use my answers to create a product requirements document (PRD) for the project.
- **What AI gave me**: AI asked me questions about the website's purpose, audience, features, design, and scope. It then organized my answers into a `PRD.md` file for my final project.
- **What I kept / changed / rejected**: I kept the `PRD.md` file because it gives me a clearer plan for building the website. I reviewed the details to make sure they matched what my tennis coach needs and what I can realistically complete.

### Interaction 2

- **What I asked**: I asked AI to help me add a copy-to-clipboard button to my dark-mode assignment.
- **What AI gave me**: AI helped me write JavaScript that copies text to the user's clipboard when the button is clicked and gives the user feedback that it was copied.
- **What I kept / changed / rejected**: I kept the clipboard feature and reviewed how the button connects to the JavaScript so I could understand how the browser's clipboard feature works.
